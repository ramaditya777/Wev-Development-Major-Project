Temperature Converter

Convert Fahrenheit

to Celsius : ℃ = ( ℉ - 32 ) / 1.8

to Kelvin : K = (( ℉ - 32 ) / 1.8 ) + 273.15

Convert Celsius

to Fahrenheit : ℉ = ( ℃ * 1.8 ) + 32

to Kelvin : K = ℃ + 273.15

Convert Kelvin

to Fahrenheit : ℉ = (( K - 273.15 ) * 1.8 ) + 32

to Celsius : ℃ = K - 273.15
